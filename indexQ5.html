Full Stack Web Development tech stack

Front End (clent-side)

HTML
HTML (HyperText Markup Language) is the code that is used to structure a web page and its content. For example, content could be structured within a set of paragraphs, a list of bulleted points, or using images and data tables.
CSS
Cascading Style Sheets (CSS) is a simple mechanism for adding style (e.g., fonts, colors, spacing) to Web documents. These pages contain information on how to learn and use CSS and on available software. They also contain news from the CSS working group.
JavaScript
A programming language used to add interactivity and functionality to web pages. Common libraries and frameworks include:

React: A JavaScript library for building user interfaces.

Angular: A TypeScript-based framework for building web applications.

Vue.js: A progressive JavaScript framework for building user interfaces.

Responsive Design
Techniques and CSS frameworks (e.g., Bootstrap, Foundation) for creating web designs that adapt to different screen sizes and devices.
AJAX
Asynchronous JavaScript and XML or other data-fetching techniques for making asynchronous requests to the server without refreshing the entire page.
Browser Developer Tools
Browsers' built-in developer tools (e.g., Chrome DevTools) for debugging and inspecting web applications.

Back-End (Server-Side)

Server-Side Languages:
Node.js:
A runtime environment for executing JavaScript on the server-side.
Python:
A versatile language often used with frameworks like Django or Flask.
Ruby:
Known for the Ruby on Rails framework.
Java:
Often used with the Spring framework.
C#:
Commonly used with ASP.NET.
Web Frameworks and Libraries:
Express.js:
A popular Node.js web application framework.
Ruby on Rails:
A full-stack web framework for Ruby.
Django:
A high-level Python web framework.
ASP.NET Core:
A cross-platform web framework for C#.
Spring Boot:
An application framework for Java.

Databases:

Relational Databases:
Such as MySQL, PostgreSQL, or Microsoft SQL Server.
NoSQL Databases:
Such as MongoDB, Cassandra, or Redis.
APIs and RESTful Services:
Creating APIs to interact with the front-end and other services.
